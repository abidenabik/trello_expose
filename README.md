# trello_expose

Ce projet accompagne une mini-conférence présentée en binôme, avec :

- La planification organisée sur Trello
- Les slides réalisées en LaTeX (Beamer) via Overleaf
- Une démonstration d’utilisation de GitHub pour la collaboration en équipe

## Contenu du dépôt

- Un tableau Trello pour suivre les tâches
- Un repo GitHub avec branches, pull requests et issues
- Les fichiers LaTeX pour générer les slides

## Objectif

Montrer comment utiliser GitHub et Trello pour organiser un mini-projet collaboratif, tout en préparant une présentation professionnelle avec LaTeX.

---

## Instructions

1. Consulter le tableau Trello pour le suivi du projet  
2. Explorer le repo GitHub, ses branches, PRs et issues  
3. Compiler les slides avec Overleaf ou TeX Live
